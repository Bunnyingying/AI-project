# Motorola 复古打字机模拟器

一款具有复古风格的打字机网页应用，让用户体验怀旧的打字乐趣，生成可交互的纸张卡片。

## 项目介绍

这款打字机模拟器灵感来源于经典的Motorola设备，结合了复古视觉风格与现代交互体验。用户可以输入文本，生成带有打字效果的纸张卡片，这些卡片支持拖拽、编辑、删除，并会自动保存到浏览器本地存储中，即使刷新页面也不会丢失。

## 功能特点

- **复古打字效果**：模拟真实打字机的打字动画与声音反馈
- **可交互纸张卡片**：
  - 生成带有日期时间戳的纸张
  - 支持拖拽移动位置
  - 双击卡片内容可进行编辑
  - 右上角删除按钮可移除卡片
- **本地存储**：所有卡片自动保存到localStorage，刷新页面不丢失
- **自适应设计**：卡片高度随内容自动调整，内容过多时显示滚动条
- **视觉细节**：包含纸张横线、胶带装饰等复古元素

## 使用方法

1. 在底部控制栏的输入框中输入文本（支持最多500个字符）
2. 点击"PRINT"按钮或按下回车键生成卡片
3. 卡片生成后会自动"弹出"并可拖拽移动
4. 双击卡片内容可进行编辑，点击其他区域保存修改
5. 点击卡片右上角的"×"按钮删除卡片

## 安装与运行

无需安装，直接将HTML文件在浏览器中打开即可使用：

1. 下载项目中的`index.html`文件
2. 用任意现代浏览器（Chrome、Firefox、Edge等）打开该文件
3. 开始使用打字机功能

## 技术栈

- HTML5：页面结构与语义化标签
- CSS3：样式设计，包括复古风格与动画效果
- JavaScript：交互逻辑实现
- Web Audio API：打字声音效果
- localStorage：本地数据存储

## 开发说明

### 核心功能实现

1. **打字效果**：通过逐字符添加与定时器实现打字动画
2. **拖拽功能**：基于鼠标事件监听实现元素拖拽
3. **本地存储**：使用localStorage API保存卡片数据，包括内容、位置和旋转角度
4. **响应式设计**：通过CSS灵活布局确保在不同设备上的良好显示

- 实现卡片分类与搜索功能
- 支持导出卡片为图片
