**AI 指令：重新设计应用配色方案，增强模块区分度**

---

## 一、问题分析

当前配色问题：
- 所有模块使用相同的背景色（`var(--bg)` 和 `var(--surface)`），区分度不足
- 边框颜色（`var(--border)`）统一，无法区分不同功能区
- 缺乏视觉层次，侧边栏、月视图、面板之间界限模糊

---

## 二、设计目标

1. **明确分区**：侧边栏、月视图、顶部栏使用不同背景色或阴影层次
2. **增强可读性**：任务卡片、感想卡片、目标列表有独立的视觉语言
3. **保持一致性**：暗色模式同步适配，所有颜色使用 CSS 变量
4. **符合现代设计**：采用微妙的层次感，而非强烈对比

---

## 三、配色方案设计

### 3.1 主色调保留
- `--primary: #2563eb`（蓝色）保持不变，作为主要操作色
- `--success: #16a34a`（绿色）保持不变，作为完成状态色

### 3.2 新增背景层次变量

```css
:root {
  /* 原有变量 */
  --bg: #ffffff;
  --surface: #f8fafc;
  --border: #e5e7eb;
  
  /* 新增层次变量 */
  --sidebar-bg: #f9fafb;      /* 侧边栏背景（比主背景稍暖） */
  --calendar-bg: #ffffff;      /* 月视图背景（纯白） */
  --card-bg: #ffffff;          /* 卡片背景（纯白） */
  --hover-bg: #f1f5f9;         /* 悬停背景 */
  
  /* 新增边框/阴影变量 */
  --border-light: #eef2f6;     /* 轻边框，用于模块分隔 */
  --border-strong: #e2e8f0;    /* 强边框，用于卡片边界 */
  --shadow-sm: 0 1px 2px rgba(0,0,0,0.05);
  --shadow-md: 0 4px 6px -1px rgba(0,0,0,0.1);
}
```

### 3.3 暗色模式变量

```css
[data-theme="dark"] {
  --bg: #0a0c10;
  --surface: #111827;
  --sidebar-bg: #0f1117;       /* 侧边栏稍暗 */
  --calendar-bg: #111827;       /* 日历背景稍亮 */
  --card-bg: #1a1e2a;           /* 卡片背景 */
  --hover-bg: #1f2937;
  --border-light: #1e293b;
  --border-strong: #2d3a4f;
  --shadow-sm: 0 1px 2px rgba(0,0,0,0.2);
  --shadow-md: 0 4px 6px -1px rgba(0,0,0,0.3);
}
```

## 四、各模块应用规则

### 4.1 整体布局

| 区域       | 背景色               | 边框                         | 阴影 |
| ---------- | -------------------- | ---------------------------- | ---- |
| 页面主体   | `var(--bg)`          | 无                           | 无   |
| 侧边栏     | `var(--sidebar-bg)`  | 右边框 `var(--border-light)` | 无   |
| 月视图区域 | `var(--calendar-bg)` | 无                           | 无   |
| 顶部栏     | `var(--bg)`          | 下边框 `var(--border)`       | 无   |

### 4.2 功能面板（侧边栏内）

| 面板         | 背景             | 边框                  | 圆角 | 阴影               |
| ------------ | ---------------- | --------------------- | ---- | ------------------ |
| 本月目标面板 | `var(--card-bg)` | `var(--border-light)` | 16px | `var(--shadow-sm)` |
| 每日感想面板 | `var(--card-bg)` | `var(--border-light)` | 16px | `var(--shadow-sm)` |

### 4.3 日期格子（月视图）

| 状态         | 背景             | 边框                      | 阴影               |
| ------------ | ---------------- | ------------------------- | ------------------ |
| 普通日期     | `var(--card-bg)` | `var(--border-light)`     | `var(--shadow-sm)` |
| 今日日期     | `var(--card-bg)` | `var(--primary)` + 外发光 | `var(--shadow-md)` |
| 非当前月日期 | `var(--surface)` | `var(--border-light)`     | 无，降低透明度     |

### 4.4 任务卡片

| 状态   | 背景        | 边框                  | 悬停效果   |
| ------ | ----------- | --------------------- | ---------- |
| 未完成 | `var(--bg)` | `var(--border-light)` | 边框变主色 |
| 已完成 | `var(--bg)` | `var(--border-light)` | 文字变灰   |

### 4.5 感想卡片

| 状态 | 背景              | 左边框   | 阴影 |
| ---- | ----------------- | -------- | ---- |
| 普通 | `var(--bg)`       | 无       | 无   |
| 悬停 | `var(--hover-bg)` | 3px 主色 | 无   |

### 4.6 目标列表项

| 状态 | 背景              | 边框                  | 圆角 |
| ---- | ----------------- | --------------------- | ---- |
| 普通 | `var(--bg)`       | `var(--border-light)` | 12px |
| 悬停 | `var(--hover-bg)` | `var(--primary)`      | 12px |

## 五、CSS 修改指令

### 5.1 替换现有颜色变量
在 `:root` 和 `[data-theme="dark"]` 中：
- 保留原有 `--bg`、`--surface`、`--border`
- 新增上述所有层次变量

### 5.2 修改 body 背景
```css
body {
  background: var(--bg);  /* 保持不变，但 --bg 值已调整 */
}
```

### 5.3 修改布局背景
```css
.layout {
  /* 不直接设置背景，让背景透出 body 的背景 */
}

.sidebar {
  background: var(--sidebar-bg);
  border-right: 1px solid var(--border-light);
}

.week {
  background: var(--calendar-bg);
}
```

### 5.4 修改面板样式
```css
.panel {
  background: var(--card-bg);
  border: 1px solid var(--border-light);
  border-radius: 16px;
  box-shadow: var(--shadow-sm);
}
```

### 5.5 修改日期格子
```css
.month-day {
  background: var(--card-bg);
  border: 1px solid var(--border-light);
  box-shadow: var(--shadow-sm);
}

.month-day.outside {
  background: var(--surface);
  opacity: 0.7;
}

.month-day.today {
  border: 2px solid var(--primary);
  box-shadow: 0 0 0 3px rgba(37, 99, 235, 0.2), var(--shadow-md);
}
```

### 5.6 修改任务卡片
```css
.task {
  background: var(--bg);
  border: 1px solid var(--border-light);
  border-radius: 10px;
}

.task:hover {
  border-color: var(--primary);
}
```

### 5.7 修改感想卡片
```css
.thought-item {
  background: var(--bg);
  border-radius: 12px;
  transition: all 0.2s;
}

.thought-item:hover {
  background: var(--hover-bg);
  transform: translateX(4px);
}
```

### 5.8 修改目标列表项
```css
.goal-list li {
  background: var(--bg);
  border: 1px solid var(--border-light);
  border-radius: 12px;
}

.goal-list li:hover {
  border-color: var(--primary);
  background: var(--hover-bg);
}
```

### 5.9 修改输入框和按钮
```css
input, textarea {
  background: var(--bg);
  border: 1px solid var(--border-light);
}

.btn.secondary {
  background: var(--surface);
  border: 1px solid var(--border-light);
}
```

## 六、效果预期

修改完成后，视觉层次将呈现：

1. **最底层**：body 背景（浅灰/深色）
2. **中层**：侧边栏（微暖/微暗）和月视图（纯白/深灰）
3. **顶层**：白色卡片（纯白/亮色），带浅边框和阴影
4. **强调层**：今日日期带蓝色边框和外发光

各模块通过**背景色差异**、**边框粗细**、**阴影深度**实现自然区分，而非依赖强烈颜色对比。
